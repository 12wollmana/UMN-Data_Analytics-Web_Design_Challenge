Welcome to Latitude!

See how latitude influences weather around the globe.

This website explores how latitude and weather are related to see if there is any correlation between the two.

Data was pulled from OpenWeatherMap to get the weather for over 500 cities at different latitudes.

Plots were generated using Matplotlib. These plots compare latitude to various weather aspects, like cloudiness and maximum temperature.

Use the links above or the images on this page to explore!
